# graphite-core

Rust core starter for Graphite.

## Modules

- Account resolution
- Transaction planning
- Confidence scoring
- Risk assessment
- Policy evaluation
- Semantic graph storage
- Simulation integrity
- Regression engine
- Plugin orchestration
- Verification coordinator

## Primary entry point

Use `GraphiteCore::verify(...)` from `src/verification.rs`.

This starter is deterministic and can be expanded into a real Solana verification core.
